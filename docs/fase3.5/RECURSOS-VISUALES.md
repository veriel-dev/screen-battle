# Recursos Visuales para Kodamon

Este documento recopila sprites, assets y recursos visuales recomendados para mejorar el apartado gráfico del juego.

## Especificaciones Técnicas Requeridas

| Elemento | Tamaño | Formato | Notas |
|----------|--------|---------|-------|
| Sprites Kodamon | 64x64 px | PNG con alpha | Pixel art |
| Fondos de batalla | 512x384 px | PNG | Escalable |
| Elementos UI | Variable | PNG/SVG | 9-slice compatible |
| Efectos VFX | 32x32 - 64x64 px | Spritesheet PNG | Animados |

## 7. Plan de Implementación

### Fase A: Assets Prioritarios (Reemplazo Inmediato)

1. **Sprites Kodamon** - Reemplazar generación procedural
   - Descargar pack de 64x64 monsters
   - Adaptar nombres a los 10 Kodamon existentes
   - Integrar en BootScene.ts

2. **Fondo de Batalla** - Un fondo neutro inicial
   - Descargar Hazy Hills o similar
   - Escalar a 512x384
   - Integrar en BattleScene.ts

3. **UI Básica** - HealthBar y botones
   - Descargar Kenney Pixel UI Pack
   - Reemplazar rectángulos por sprites
   - Actualizar componentes UI

### Fase B: Mejoras Visuales

4. **VFX de Ataques**
   - Descargar efectos por tipo
   - Integrar en BattleEffects.ts
   - Mapear a movimientos

5. **Fondos por Tipo**
   - Múltiples fondos según tipos
   - Sistema de selección aleatoria

6. **Iconos de Tipo**
   - Crear/descargar iconos 16x16
   - Mostrar en UI de batalla

### Fase C: Polish

7. **Animaciones de Criaturas**
   - Idle, ataque, daño, desmayo
   - Spritesheets animados

8. **Efectos de Partículas**
   - Sistema de partículas Phaser
   - Efectos ambientales
